## Acceptance Gates (The "Zero-Scroll" Certification)

For **Laptop Standard (1366x768)**, the following MUST be visible simultaneously:
1.  **Tariff Input** (at least 3 rows)
2.  **Live Summary Panel** (Total & Margin)
3.  **"Add to Offer" Button** (Primary CTA)
4.  **Header Navigation**

If the user has to scroll to find the "Add" button while selecting a tariff, the test **FAILS**.
